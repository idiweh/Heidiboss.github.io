Plant minigame (AR)

This is an introductionary project made as an assignment in university.

The goal of the project was to learn how to develop and build an AR game using Unity game engine. The point of the game is to get the plant fully nutritioned and watered, so the plant will grow. Very simple. I practiced the AR features by adding the options to rotate and scale plant. The plane recognition does not work like I wanted it to, but it was also a bit dark when I filmed the video.

I will definately dive deeper into AR world later.
